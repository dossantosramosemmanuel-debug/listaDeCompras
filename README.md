# Lista de Compras

Este projeto contém uma aplicação simples de lista de compras feita com HTML, CSS e JavaScript.

## Arquivos

- `index.html` - estrutura da página
- `styles.css` - estilos visuais
- `script.js` - lógica para adicionar, marcar e remover itens

## Como usar

1. Abra o arquivo `index.html` em um navegador.
2. Digite um item na caixa de entrada e informe o preço.
3. Clique em `Adicionar` ou pressione Enter.
4. Marque itens como concluídos com o checkbox.
5. O painel exibe quanto você pagaria pelos itens marcados como OK.
6. Remova itens com o botão `Remover`.
7. Use `Limpar concluídos` ou `Limpar tudo` para manter a lista organizada.

## Recursos

- Persistência de itens usando `localStorage`
- Ações de adicionar, concluir e remover
- Layout responsivo para dispositivos móveis
